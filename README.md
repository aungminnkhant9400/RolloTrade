# RolloTrade

RolloTrade is a mobile-first trading video study guide. It turns long trading
videos into readable field notes, step-by-step execution guides, checklists,
timestamp maps, and practice drills.

## Local Use

Open `index.html` in a browser. The site is static and has no build step.

## Structure

- `index.html` - page structure
- `styles.css` - responsive layout and print styles
- `app.js` - lesson data and rendering logic

## Next Videos

Add new lessons to the `lessons` array in `app.js`. Each lesson supports:

- summary
- tags
- quick cards
- execution steps
- checklist items
- drills
- risk notes
- timestamp notes
